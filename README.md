실행 방법

1. 외부 라이브러리 설치
폴더에서 터미널을 열고 아래 명령어를 입력하여 필요한 도구들을 설치한다.
> pip install -r requirements.txt

2. 환경 변수 설정
프로젝트 폴더에 `.env`라는 파일에 본인의 키 정보를 적는다. (필요시 변경, 이미 적혀있음)
> GH_ACCOUNT = 
> GH_APPKEY = 
> GH_APPSECRET = 

3. 프로그램 실행
터미널에 아래 명령어를 입력한다.
> python main.py
